# Rishabh Tiwari — Resume Website

Single-page, responsive resume website built with HTML, CSS, and vanilla JavaScript.

## Run locally

```bash
cd "/Users/rishabhtiwari/Desktop/rishabh_portfolio"
python3 -m http.server 5173
```

Open `http://127.0.0.1:5173/`.

## Folder structure

- `index.html` — page content
- `styles.css` — light/dark theme styling
- `script.js` — theme toggle + small UX behaviors
- `assets/` — resume PDF and images
- `certificates/` — certificate PDFs
